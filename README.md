# Expense-Management-Dashboard-Excel-VBA-Automation-Project.

Project Overview

This project demonstrates how Microsoft Excel, Pivot Tables, VBA Automation, and AI-assisted development can be combined to transform raw expense data into an interactive business dashboard.

The objective of this project was to automate data preparation, analysis, and reporting processes while providing decision-makers with clear insights into organizational expenses, budgets, approvals, and spending trends.

To maintain data privacy and confidentiality, a secondary/sample dataset was used throughout the project.

Project Objectives
Automate repetitive Excel reporting tasks using VBA.
Clean and structure raw expense data.
Generate meaningful business insights.
Build an interactive dashboard for expense monitoring.
Improve reporting efficiency and decision-making.

VBA Automation
